# 功能说明
工作队列(workqueue)延时跟踪工具

使用说明:
```
Usage: sysak workqlatency [options]
Options:
    --time/-t		specify the monitor period(s), default=10s
    --help/-h		help info
example: sysak workqlatency -t 100  #trace work runtime and latency statistics
```
