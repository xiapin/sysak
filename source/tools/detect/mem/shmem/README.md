# 功能说明
检测shmem泄漏时对应的使用进程
# 使用
```
使用sysak shmem -h可查看所有可用的参数
 -h   帮助信息
 -t   诊断某个时间段的shmem对应的使用进程
for example:
sysak shmem -t 5
```
