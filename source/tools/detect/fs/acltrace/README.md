# 功能说明
ext4文件系统acl跟踪工具

使用说明:
```Usage: sysak acltrace [options]
Options:
    --time/-t     specify the monitor period(s), default=5000s
    --help/-h     help info
```

使用示例：sysak acltrace -t 60 监控60‘s内设置acl情况，其中会申请kmalloc_64,若持续大量的申请，slab内存使用量也会持续增加，最终导致系统可以内存越来越少。
